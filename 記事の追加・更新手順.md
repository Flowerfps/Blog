# 記事の追加・更新手順メモ

ブログサイトの記事は`main.js`ファイルの`articles`配列を編集すれば追加・変更・削除できる。

## クイックスタート

1. `main.js`を開く
2. `const articles = [`の配列内に新しい記事を追加
3. 保存してブラウザで確認

簡単な例：
```javascript
const articles = [
    // 既存の記事...
    {
        id: 6,
        // ...
    },
    // ここに新しい記事を追加
    {
        id: 7,
        title: '記事のタイトル',
        excerpt: '記事の説明',
        content: `
# タイトル

本文を書く
        `,
        category: 'カテゴリー',
        tags: ['タグ1', 'タグ2'],
        date: '2024-02-15',
        readTime: 5,
        thumbnail: ''
    }
];
```

保存したらブラウザで`index.html`を開いて、F5でリロードして確認。

## 記事を追加する

`main.js`の2行目あたりにある`const articles = [`の配列に新しい記事オブジェクトを追加する。

既存の記事の後に追加する感じで：

```javascript
const articles = [
    {
        id: 1,
        title: '既存の記事',
        // ...
    },
    // ここに新しい記事を追加
    {
        id: 7,  // 既存の最大ID + 1にする
        title: '新しい記事のタイトル',
        excerpt: '記事の説明文。一覧ページで表示される。',
        content: `
# 記事のタイトル

本文を書く。

## 見出し2

段落。

### 見出し3

- リスト1
- リスト2

\`\`\`javascript
// コードブロック
const code = 'example';
\`\`\`

**太字**や*斜体*も使える。
        `,
        category: 'カテゴリー名',
        tags: ['タグ1', 'タグ2'],
        date: '2024-02-15',
        readTime: 5,
        thumbnail: ''
    }
];
```

必須項目：
- id: 重複しない番号
- title: タイトル
- excerpt: 説明文（50-150文字くらい）
- content: 本文（Markdown形式）
- category: カテゴリー
- tags: タグの配列
- date: 日付（YYYY-MM-DD形式）
- readTime: 読了時間（分）
- thumbnail: サムネイル

保存してブラウザで確認。

## 記事を編集する

編集したい記事の`id`を見つけて、内容を変更する。

```javascript
{
    id: 1,  // IDは変えない
    title: '変更後のタイトル',
    excerpt: '変更後の説明',
    content: `
# 変更後の内容

更新した内容。
    `,
    category: '変更後のカテゴリー',
    tags: ['新しいタグ'],
    date: '2024-02-20',
    readTime: 10,
    thumbnail: ''
}
```

保存してブラウザでリロードして確認。

## 記事を削除する

削除したい記事の`id`を見つけて、記事全体（`{`から`}`まで）を削除する。

```javascript
const articles = [
    {
        id: 1,
        // ...
    },
    {
        id: 2,
        // ...
    },
    // この記事を削除
    // {
    //     id: 3,
    //     ...
    // },
    {
        id: 4,
        // ...
    }
];
```

カンマの位置に注意。配列の最後の記事を削除する場合は、前の記事のカンマも削除する。

## よく使う項目

| 項目 | 説明 | 例 |
|------|------|-----|
| id | 記事のID（重複しない番号） | 7 |
| title | タイトル | 'React入門' |
| excerpt | 説明（50-150文字） | 'Reactの基本を学ぶ' |
| category | カテゴリー | 'React' |
| tags | タグ（配列） | ['React', 'JavaScript'] |
| date | 日付（YYYY-MM-DD） | '2024-02-15' |
| readTime | 読了時間（分） | 5 |
| thumbnail | サムネイル | '' |

## 注意事項

- カンマの位置：配列の最後の記事以外は、記事の後にカンマが必要
- IDの重複：同じIDを2つ使わない
- 引用符：文字列は必ず`'`または`"`で囲む
- 日付形式：必ず`YYYY-MM-DD`形式で書く

## 記事データの構造

```javascript
{
    id: 数値,
    title: '文字列',
    excerpt: '文字列',
    content: `文字列`,
    category: '文字列',
    tags: ['配列'],
    date: 'YYYY-MM-DD',
    readTime: 数値,
    thumbnail: '文字列'
}
```

### 各項目の説明

id: 記事を識別する番号。1, 2, 3...と重複しないようにする。

title: 記事のタイトル。例：`'React Hooks完全ガイド'`

excerpt: 記事の説明文。一覧ページで表示される。50-150文字くらいがいい。例：`'React Hooksの基本から応用まで解説します'`

content: 記事の本文。Markdown形式で書ける。バッククォート3つ（\`\`\`）で囲む。改行はそのまま反映される。

Markdown記法の例：
```markdown
# 見出し1
## 見出し2
### 見出し3

**太字**
*斜体*

- リスト項目1
- リスト項目2

\`\`\`javascript
// コードブロック
const code = 'example';
\`\`\`
```

category: 記事のカテゴリー。例：`'React'`, `'TypeScript'`, `'CSS'`など。既存のカテゴリーを使うか、新しいカテゴリーを作成できる。

tags: 記事のタグ。配列形式で複数指定できる。例：`['React', 'JavaScript', 'Hooks']`

date: 記事の公開日。YYYY-MM-DD形式で指定。例：`'2024-02-15'`

readTime: 記事の読了時間（分）。数値で指定。例：`8`（8分で読める）

thumbnail: 記事のサムネイル。絵文字でもいいし、空文字でもいい。

## よくある質問

Q: 記事を追加したのに表示されない
A: 以下を確認：
- ファイルを保存したか
- ブラウザをリロードしたか（F5キー）
- JavaScriptの構文エラーがないか（カンマの位置、括弧の閉じ忘れなど）

Q: カテゴリーやタグを追加したい
A: `category`や`tags`に新しい名前を指定するだけで自動的に追加される。既存のカテゴリーやタグと統一したい場合は、同じ名前を使う。

Q: 記事の本文で改行が反映されない
A: `content`内では、空行を入れると段落として認識される。改行だけでは反映されないので、空行を入れる。

Q: コードブロックを追加したい
A: バッククォート3つ（\`\`\`）で囲む：
```markdown
\`\`\`javascript
// コードをここに書く
\`\`\`
```

Q: 日付の形式を間違えた
A: 必ず`YYYY-MM-DD`形式で書く。例：`'2024-02-15'`

Q: 記事の順番を変更したい
A: `articles`配列内の記事の順番を入れ替えると、記事一覧の表示順が変わる。新しい記事を上に表示したい場合は、配列の先頭に追加する。

Q: エラーが出る
A: よくある原因：
- カンマ（`,`）の位置が間違っている
- 引用符（`'`や`"`）が閉じられていない
- 括弧（`{`や`[`）が閉じられていない
- `id`が重複している

エラーメッセージを確認して、該当箇所を修正する。

## テンプレート

新しい記事を追加するときに使えるテンプレート：

```javascript
{
    id: 0,  // 既存の最大ID + 1に変更
    title: '記事のタイトルをここに書く',
    excerpt: '記事の簡単な説明を50-150文字程度で書く',
    content: `
# 記事のタイトル

ここに記事の本文を書く。

## 見出し2

段落を書く。

### 見出し3

- リスト項目1
- リスト項目2

\`\`\`javascript
// コードブロックの例
const example = 'Hello World';
\`\`\`

**太字**や*斜体*も使える。
    `,
    category: 'カテゴリー名',
    tags: ['タグ1', 'タグ2'],
    date: '2024-02-15',
    readTime: 5,
    thumbnail: ''
}
```

## Markdown記法一覧

記事の本文（`content`）で使えるMarkdown記法：

| 記法 | 結果 |
|------|------|
| # 見出し1 | 大きな見出し |
| ## 見出し2 | 中くらいの見出し |
| ### 見出し3 | 小さな見出し |
| **太字** | **太字** |
| *斜体* | *斜体* |
| - リスト | 箇条書き |
| `コード` | インラインコード |
| \`\`\`言語名 | コードブロック |

## まとめ

1. 記事を追加: `main.js`の`articles`配列に新しい記事オブジェクトを追加
2. 記事を編集: 既存の記事オブジェクトの内容を変更
3. 記事を削除: 記事オブジェクト全体を削除
4. 保存して確認: ファイルを保存し、ブラウザで確認
