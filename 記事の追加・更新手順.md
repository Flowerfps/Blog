## クイックスタート（3ステップ）

### ステップ1: main.jsを開く
プロジェクトフォルダ内の`main.js`ファイルを開きます。

### ステップ2: 記事を追加
ファイルの2行目あたりにある`const articles = [`の配列内に、新しい記事を追加します。

追加する位置：
```javascript
const articles = [
    // 既存の記事...
    {
        id: 6,
        // ... 既存の記事6
    },
    // ここに新しい記事を追加
    {
        id: 7,
        title: '私の最初の記事',
        excerpt: 'これは私が書いた最初の記事です。',
        content: `
# 私の最初の記事

こんにちは！これは私が書いた最初の記事です。

## 学んだこと

今日は以下のことを学びました：

- HTMLの基本
- CSSのスタイリング
- JavaScriptの基礎

これからも頑張ります！
        `,
        category: '学習記録',
        tags: ['初心者', '学習'],
        date: '2024-02-15',
        readTime: 3,
        thumbnail: ''
    }
];
```

### ステップ3: 保存して確認
1. ファイルを保存（Ctrl+S / Cmd+S）
2. ブラウザで`index.html`を開く
3. ページをリロード（F5キー）
4. 新しい記事が表示されることを確認

## 記事の追加方法

### ステップ1: main.jsファイルを開く
1. プロジェクトフォルダ内の`main.js`ファイルを開きます
2. ファイルの先頭部分（2行目あたり）に`const articles = [`という行があります

### ステップ2: 新しい記事を追加
既存の記事の後に、新しい記事オブジェクトを追加します。

例：新しい記事を追加する場合

```javascript
const articles = [
    // 既存の記事1
    {
        id: 1,
        title: '既存の記事タイトル',
        // ... 既存の記事データ
    },
    // 既存の記事2
    {
        id: 2,
        title: '既存の記事タイトル2',
        // ... 既存の記事データ
    },
    // ここに新しい記事を追加
    {
        id: 7,  // 既存の最大ID + 1（現在は6までなので7）
        title: '私の新しい記事タイトル',
        excerpt: 'この記事の簡単な説明文をここに書きます。記事一覧ページで表示されます。',
        content: `
# 記事のタイトル

ここに記事の本文を書きます。

## 見出し2

段落を書くことができます。

### 見出し3

- リスト項目1
- リスト項目2
- リスト項目3

\`\`\`javascript
// コードブロックの例
const example = 'Hello World';
console.log(example);
\`\`\`

**太字**や*斜体*も使えます。
        `,
        category: 'カテゴリー名',
        tags: ['タグ1', 'タグ2', 'タグ3'],
        date: '2024-02-15',  // YYYY-MM-DD形式
        readTime: 5,  // 読了時間（分）
        thumbnail: ''  // 絵文字またはアイコン
    }
];
```

### ステップ3: 必須項目の確認
以下の項目は必ず記入してください：
- id: ユニークな番号（既存の記事と重複しないように）
- title: 記事のタイトル
- excerpt: 記事の抜粋（50-150文字程度推奨）
- content: 記事の本文
- category: カテゴリー名
- tags: タグの配列（複数可）
- date: 日付（YYYY-MM-DD形式）
- readTime: 読了時間（分）
- thumbnail: サムネイル（絵文字推奨）

### ステップ4: 保存して確認
1. ファイルを保存（Ctrl+S / Cmd+S）
2. ブラウザで`index.html`を開いて確認
3. 新しい記事が一覧に表示されることを確認

## 記事の編集・更新方法

### ステップ1: 編集したい記事を見つける
`main.js`ファイル内で、編集したい記事の`id`を確認します。

### ステップ2: 記事データを編集
該当する記事オブジェクト内の各項目を編集します。

例：タイトルと内容を変更する場合

```javascript
{
    id: 1,  // IDは変更しない
    title: '変更後のタイトル',  // タイトルを変更
    excerpt: '変更後の抜粋',  // 抜粋を変更
    content: `
# 変更後の内容

記事の内容を更新しました。
    `,
    // 他の項目も必要に応じて変更
    category: '変更後のカテゴリー',
    tags: ['新しいタグ1', '新しいタグ2'],
    date: '2024-02-20',  // 更新日を変更
    readTime: 10,
    thumbnail: ''
}
```

### ステップ3: 保存して確認
1. ファイルを保存
2. ブラウザで確認（ページをリロード）

## 記事の削除方法

### ステップ1: 削除したい記事を見つける
`main.js`ファイル内で、削除したい記事の`id`を確認します。

### ステップ2: 記事を削除
該当する記事オブジェクト全体を削除します。

例：id: 3の記事を削除する場合

```javascript
const articles = [
    {
        id: 1,
        // ... 記事1のデータ
    },
    {
        id: 2,
        // ... 記事2のデータ
    },
    // この記事全体を削除
    // {
    //     id: 3,
    //     title: '削除する記事',
    //     ...
    // },
    {
        id: 4,
        // ... 記事4のデータ
    }
];
```

注意：カンマ（`,`）の位置に注意してください。配列の最後の記事を削除する場合は、前の記事のカンマも削除してください。

## よく使う項目

| 項目 | 説明 | 例 |
|------|------|-----|
| id | 記事のID（重複しない番号） | 7 |
| title | 記事のタイトル | 'React入門' |
| excerpt | 記事の説明（50-150文字） | 'Reactの基本を学びます' |
| category | カテゴリー | 'React' |
| tags | タグ（配列） | ['React', 'JavaScript'] |
| date | 日付（YYYY-MM-DD） | '2024-02-15' |
| readTime | 読了時間（分） | 5 |
| thumbnail | サムネイル（絵文字） | '' |

## 注意事項

1. カンマの位置: 配列の最後の記事以外は、記事の後にカンマ（`,`）が必要です
2. IDの重複: 同じIDを2つ使わないでください
3. 引用符: 文字列は必ず引用符（`'`または`"`）で囲んでください
4. 日付形式: 日付は必ず`YYYY-MM-DD`形式で書いてください

## 記事データの構造

各記事は以下の構造になっています：

```javascript
{
    id: 数値,              // 必須：ユニークなID（1, 2, 3...）
    title: '文字列',        // 必須：記事のタイトル
    excerpt: '文字列',      // 必須：記事の抜粋（一覧表示用）
    content: `文字列`,      // 必須：記事の本文（Markdown形式）
    category: '文字列',     // 必須：カテゴリー名
    tags: ['配列'],        // 必須：タグの配列
    date: 'YYYY-MM-DD',    // 必須：日付（例：'2024-02-15'）
    readTime: 数値,        // 必須：読了時間（分）
    thumbnail: '絵文字'     // 必須：サムネイル（絵文字推奨）
}
```

### 各項目の詳細説明

#### id（必須）
- 記事を識別するためのユニークな番号
- 数値で指定（1, 2, 3...）
- 既存の記事と重複しないように注意

#### title（必須）
- 記事のタイトル
- 例：`'React Hooks完全ガイド'`

#### excerpt（必須）
- 記事の簡単な説明文
- 記事一覧ページで表示される
- 50-150文字程度が推奨
- 例：`'React Hooksの基本から応用まで解説します'`

#### content（必須）
- 記事の本文
- Markdown形式で記述可能
- バッククォート3つ（\`\`\`）で囲む
- 改行はそのまま反映されます

Markdown記法の例：
```markdown
# 見出し1
## 見出し2
### 見出し3

**太字**
*斜体*

- リスト項目1
- リスト項目2

\`\`\`javascript
// コードブロック
const code = 'example';
\`\`\`
```

#### category（必須）
- 記事のカテゴリー
- 例：`'React'`, `'TypeScript'`, `'CSS'`など
- 既存のカテゴリーを使うか、新しいカテゴリーを作成可能

#### tags（必須）
- 記事のタグ（配列形式）
- 複数のタグを指定可能
- 例：`['React', 'JavaScript', 'Hooks']`

#### date（必須）
- 記事の公開日
- YYYY-MM-DD形式で指定
- 例：`'2024-02-15'`

#### readTime（必須）
- 記事の読了時間（分）
- 数値で指定
- 例：`8`（8分で読める）

#### thumbnail（必須）
- 記事のサムネイル
- 絵文字を推奨
- 例：`''`, `''`, `''`など

## よくある質問

### Q1: 記事を追加したのに表示されない
A: 以下の点を確認してください：
- ファイルを保存したか
- ブラウザをリロードしたか（F5キーまたはCtrl+R / Cmd+R）
- JavaScriptの構文エラーがないか（カンマの位置、括弧の閉じ忘れなど）

### Q2: カテゴリーやタグを追加したい
A: `category`や`tags`に新しい名前を指定するだけで、自動的に追加されます。既存のカテゴリーやタグと統一したい場合は、同じ名前を使いましょう。

### Q3: 記事の本文で改行が反映されない
A: `content`内では、空行を入れると段落として認識されます。改行だけでは反映されないため、空行を入れてください。

### Q4: コードブロックを追加したい
A: バッククォート3つ（\`\`\`）で囲みます：
```markdown
\`\`\`javascript
// コードをここに書く
\`\`\`
```

### Q5: 日付の形式を間違えた
A: 必ず`YYYY-MM-DD`形式で記入してください。例：`'2024-02-15'`

### Q6: 記事の順番を変更したい
A: `articles`配列内の記事の順番を入れ替えると、記事一覧の表示順が変わります。新しい記事を上に表示したい場合は、配列の先頭に追加してください。

### Q7: エラーが出る
A: よくあるエラーの原因：
- カンマ（`,`）の位置が間違っている
- 引用符（`'`や`"`）が閉じられていない
- 括弧（`{`や`[`）が閉じられていない
- `id`が重複している

エラーメッセージを確認して、該当箇所を修正してください。
