# 🚀 クイックスタートガイド

## 記事を追加する3ステップ

### ステップ1: `main.js`を開く
プロジェクトフォルダ内の`main.js`ファイルを開きます。

### ステップ2: 記事を追加
ファイルの2行目あたりにある`const articles = [`の配列内に、新しい記事を追加します。

**追加する位置：**
```javascript
const articles = [
    // 既存の記事...
    {
        id: 6,
        // ... 既存の記事6
    },
    // 👇 ここに新しい記事を追加
    {
        id: 7,
        title: '私の最初の記事',
        excerpt: 'これは私が書いた最初の記事です。',
        content: `
# 私の最初の記事

こんにちは！これは私が書いた最初の記事です。

## 学んだこと

今日は以下のことを学びました：

- HTMLの基本
- CSSのスタイリング
- JavaScriptの基礎

これからも頑張ります！
        `,
        category: '学習記録',
        tags: ['初心者', '学習'],
        date: '2024-02-15',
        readTime: 3,
        thumbnail: '📚'
    }
];
```

### ステップ3: 保存して確認
1. ファイルを保存（Ctrl+S / Cmd+S）
2. ブラウザで`index.html`を開く
3. ページをリロード（F5キー）
4. 新しい記事が表示されることを確認！

---

## 📝 記事を編集する

`main.js`内で、編集したい記事の`id`を見つけて、内容を変更します。

**例：タイトルを変更**
```javascript
{
    id: 1,
    title: '変更後のタイトル',  // 👈 ここを変更
    excerpt: '...',
    // ...
}
```

---

## 🗑️ 記事を削除する

削除したい記事の`id`を見つけて、記事全体（`{`から`}`まで）を削除します。

**注意：** カンマ（`,`）の位置に注意してください！

---

## 💡 よく使う項目

| 項目 | 説明 | 例 |
|------|------|-----|
| `id` | 記事のID（重複しない番号） | `7` |
| `title` | 記事のタイトル | `'React入門'` |
| `excerpt` | 記事の説明（50-150文字） | `'Reactの基本を学びます'` |
| `category` | カテゴリー | `'React'` |
| `tags` | タグ（配列） | `['React', 'JavaScript']` |
| `date` | 日付（YYYY-MM-DD） | `'2024-02-15'` |
| `readTime` | 読了時間（分） | `5` |
| `thumbnail` | サムネイル（絵文字） | `'⚛️'` |

---

## ⚠️ 注意事項

1. **カンマの位置**: 配列の最後の記事以外は、記事の後にカンマ（`,`）が必要です
2. **IDの重複**: 同じIDを2つ使わないでください
3. **引用符**: 文字列は必ず引用符（`'`または`"`）で囲んでください
4. **日付形式**: 日付は必ず`YYYY-MM-DD`形式で書いてください
